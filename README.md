# BG Estate Price Prediction

## Overview
This project aims to predict the prices of real estate properties in Bulgaria using various machine learning models. It leverages historical data, property features, and advanced analytics to provide accurate pricing predictions.

## Features
- Predictive modeling of real estate prices
- Interactive dashboard for data visualization
- Support for multiple machine learning algorithms
- Easy-to-use interface for end-users

## Machine Learning Models
- Linear Regression
- Random Forest
- Gradient Boosting
